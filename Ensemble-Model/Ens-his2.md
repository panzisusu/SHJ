### 2. 卡片背景科技感優化（淡灰電路網鋪底）
* **使用者：** 剛剛集成模型背景，可以改成類似這樣有點科技感嗎？大概這張圖 1/16 的大小，然後很淡的灰色即可（提供機房伺服器與電路走線的藍圖背景）。我們現在先在本機做，展示別上傳。
* **AI 助理：**
  * 為了防範未 URL 編碼之 SVG 在 Chrome 中解析失敗導致黑底黑字，編寫 Python 腳本將一組「1分3、合為1」的技術電路走線網絡 SVG 進行 URL 編碼。
  * 在 `app.js` 的 `bgStyle.background` 屬性中，將此 URL-encoded SVG 以 `80px 80px repeat` 的形式鋪設在宣紙漸層上方，呈現出極其淡雅（`#d8d3c5`）的電路底紋效果，科技感與水墨風完美相融。

### 3. 將卡片中央圖示替換為「一分三合一」電路 SVG
* **使用者：** 還不錯啊，不過細部我想改一下：A.背景的走線再粗一點點（粗 1.5 倍）。B.集成模型的圖示，也可以改成那個走線結構嗎？也在本機測試。

  * **客製化圖示**：修改 `index.html` 與 `index-zh.html` 卡片模板以支援 `project.customIcon` 的 Vue `v-html` 渲染。在 `app.js` 中新增 customIcon 欄位，放入「金色輸入源 -> 三分路經 -> 硃砂紅輸出源」的 2:1 技術走線 SVG，並在 `styles.css` 中設定寬 `120px`、高 `60px`。

### 4. 流程箭頭（Flowchart Arrows）設計與尺寸擴大
* **使用者：** 
  1. SVG 中間的線段可以變成往右的箭頭嗎？有點流程那個味道。
  2. 那個圖可以粗一倍，SVG 圖示本身高度 OK，寬度拉長一倍。
* **AI 助理：**
  * **加入流程箭頭**：在 SVG 內定義了 `<marker>` 金色與紅色箭頭。左側金色實線分叉處加上金色箭頭，右側紅色虛線匯聚處加上紅色箭頭，使其極具「前向流動」的 flowchart 質感。
  * **圖示大氣化**：將首頁 CSS 中 `.custom-card-icon svg` 的尺寸從 `120px` 拓寬至 **`180px`**，高度提升至 **`90px`**（整體放大 1.5 倍），同時將線段寬度加粗一倍（`5.6px`），使圖案更顯壯觀、平衡。
