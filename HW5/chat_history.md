## 🛠️ 核心功能模組與技術細節

### 1. 機器學習十大演算法看板
* **數據驅動 UI**：所有演算法資料保存在 `app.js` 的 `algorithms` 陣列中，透過 DOM 操控動態生成。
* **搜尋與篩選邏輯**：使用 JavaScript 的 `filter` 方法，動態配對演算法中文名、英文名、詳細原理與應用案例，配合 `classList` 切換顯示狀態。

### 2. 2D 互動式模擬沙盒
* **線性迴歸 (Linear Regression)**
  - 公式：$Y = \beta_0 + \beta_1 X + \epsilon$
  - 運算方式：利用最小平方法（OLS）求解斜率 $\beta_1$ 與截距 $\beta_0$，繪製出趨勢線。
* **K 最近鄰 (KNN)**
  - 運算方式：計算滑鼠游標（或預測網格點）與畫布上所有點的歐氏距離，篩選前 $K$ 個最近點進行投票。
  - 決策平面：將畫布切分為 $50 \times 40$ 個網格單元，即時運算每個單元中心的預測類別，以極高能效渲染半透明背景色。
* **K-Means 分群 (Clustering)**
  - 運算方式：
    1. 隨機選擇數據點作為群集中心。
    2. 分配步驟（Assign）：將點劃分給距離最近的中心。
    3. 更新步驟（Update）：計算各群組內點的 X/Y 平均座標，並將中心移過去。
  - 網頁支援點擊一步步執行，可以直觀觀察聚類中心收斂過程。
* **支持向量機 (SVM)**
  - 運算方式：使用次梯度下降法（Pegasos 算法變形）求解 $\min \frac{1}{2} \|w\|^2 + C \sum \max(0, 1 - y_i(w^T x_i + b))$。
  - 繪製了最大化邊界虛線，並對落在邊界上或內部的點（支持向量）加上發光白圈。

### 3. AI 助教聊天機器人
* **雙模式機制**：
  - **本地模式**：使用關鍵字正規表示法匹配，免金鑰回答基本的演算法與沙盒操作問題。
  - **Gemini AI 模式**：讀取 `localStorage` 中儲存的 Gemini API Key，透過 `fetch` 發送非同步 POST 請求至 `gemini-1.5-flash` 節點。
* **記憶與引導**：保留最近 12 輪的對話記錄，在發送給 Gemini 時一併作為 Context，實現上下文連續對話。

### 未來如何在本地修改並同步？
如果您在本地（您的電腦上）修改了 `index.html`、`style.css` 或 `app.js`，可以使用以下 Git 指令將其推送到 GitHub，線上網站會在一分鐘內自動更新：

```powershell
# 1. 將所有異動檔案加入暫存區
git add .

# 2. 建立提交紀錄，填寫修改說明
git commit -m "更新說明（例如：修改了沙盒的KNN顏色）"

# 3. 推送到 GitHub 遠端倉庫的 main 分支
git push
```
