分群演算法大講堂

1. K-Means 與 K-Medoids (PAM) 演算法

兩者皆為**劃分式分群 (Partitioning Clustering)**，目標是最小化各點到群心的內誤差平方和。

核心差異比較：
  • K-Means 中心點 (Centroid)： 群心是群內所有點的幾何平均值。因為是平均值，此點**可不存在於真實資料集**中。其容易受到極端值 (Outliers) 影響。
  • K-Medoids 中心點 (Medoid)： 強制中心點必須是**真實存在的數據點**，即群內與其他所有點的距離和（如 $L_1$ 曼哈頓距離）最小的點。這使得它比 K-Means 更加**耐受離群值 (Robust to Outliers)**。
K-Means 目標損失函數：
$$J = \sum_{k=1}^{K} \sum_{i \in C_k} ||x_i - \mu_k||^2$$

在本沙箱中，K-Means 用「十」字形虛線圓環代表群心，而 K-Medoids 的中心則會直接鎖定在某個實心點上，並隨每一次迭代進行轉移。

2. 密度分群法 (DBSCAN)

DBSCAN 著重於資料的分佈密度，可自動偵測噪點，且能處理任意複雜形狀的群聚。

運作核心概念：
  • 核心點 (Core Point)： 在半徑 $\epsilon$ 內，含有至少 $MinPts$ 個數據點（包括自己）。
  • 邊界點 (Border Point)： 落在核心點的 $\epsilon$ 鄰域內，但其自身的鄰域點數少於 $MinPts$。
  • 噪聲點 (Noise Point)： 既不是核心點，也不是邊界點的孤立數據。

DBSCAN 在處理 Concentric Circles、Two Moons、Spiral 等「非凸形 (Non-convex)」結構時表現完美。本沙箱會動態以紅點標記 Core Point，並將無法歸類的點視為灰色噪聲點。

3. 混合高斯模型 (GMM) 與期望最大化 (EM) 演算法

GMM 假設所有數據點是由多個具有不同均值 $\mu_k$ 和協方差矩陣 $\Sigma_k$ 的高斯分佈混合產生的。

期待最大化 (EM) 流程：
  • E-step (Expectation)： 根據當前參數計算每個數據點屬於第 $k$ 個高斯分佈的響應度 (Responsibility) $\gamma_{ik}$： $$\gamma_{ik} = \frac{\pi_k \mathcal{N}(x_i | \mu_k, \Sigma_k)}{\sum_{j=1}^{K} \pi_j \mathcal{N}(x_i | \mu_j, \Sigma_j)}$$
  • M-step (Maximization)： 使用 $\gamma_{ik}$ 作為權重，重新計算並更新每個高斯分佈的均值 $\mu_k$、協方差矩陣 $\Sigma_k$ 與混合權重 $\pi_k$。

相較於 K-Means 的硬分配 (Hard Assignment)，GMM 是一種軟分配 (Soft Assignment)，點的顏色是多種群色混合的漸層。沙箱會繪製出高斯的 $1\sigma, 2\sigma$ 擴散協方差橢圓，直觀呈現高斯分佈的形狀與方向。

4. 核函數分群 (Kernel K-Means) 與譜分群 (Spectral)

當數據在高維空間中是非線性可分時，我們可以使用核函數將點映射到高維希爾伯特空間，以使它們線性可分。

RBF 核函數公式：
$$K(x_i, x_j) = \exp(-\gamma ||x_i - x_j||^2)$$
高維空間距離評估：

點 $x_i$ 到高維空間群心 $c_k$ 的距離平方可完全由核矩陣表示：

$$d_\Phi(x_i, c_k)^2 = K(x_i, x_i) - \frac{2}{N_k}\sum_{j \in C_k} K(x_i, x_j) + \frac{1}{N_k^2}\sum_{j, l \in C_k} K(x_j, x_l)$$

這是譜分群 (Spectral Clustering) 的核近似實現。當遭遇同心圓 (Circles) 或螺旋線 (Spiral) 時，普通的 K-Means 會無能為力，但 Kernel K-Means 透過 RBF 核函數，能以超凡的非線性分割能力完美分離非凹形邊界！
